OWF 2011 "Adopt devops philosophy on your projects" Talk
=========================================================================

This is the ShowOff source for my OWF 2011 talk

  $ git clone git://github.com/theodo/OpenWorldForum2011-DevOps.git
  $ cd OpenWorldForum2011-DevOps
  $ showoff serve

Requirements
===================

You need ShowOff to run the presentation.  You can get it thusly:

  $ gem install showoff
